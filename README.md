# Smart Bookmark Manager

A modern and responsive Smart Bookmark Manager built using HTML, CSS, and Vanilla JavaScript that allows users to save, organize, and manage their favorite website links easily.

This project is inspired by the bookmark system used in Google Chrome and demonstrates how browser-based storage can be used to build a simple productivity tool.



# ✨ Features
➕ Add Bookmarks

Users can add a new bookmark by entering:

Website Name

Website URL

Category

🌍 Automatic Website Icon

The app automatically fetches the website favicon based on the URL so bookmarks look visually identifiable.

🗂 Categories

Bookmarks can be organized into categories such as:
Work
Learning
Social
Entertainment
This helps keep bookmarks structured and easy to manage.

🔍 Search Bookmarks

A real-time search feature allows users to quickly find bookmarks by name or URL.

🗑 Delete Bookmarks

Users can remove any bookmark with a delete button.

💾 Local Storage

All bookmarks are stored in the browser using Local Storage, which means:

Data stays saved after page refresh

No backend or database required

🛠 Technologies Used

HTML5
CSS3
JavaScript (Vanilla JS)
Local Storage API
